# Hypixel Stat Python

Application Dash pour afficher les statistiques Hypixel.

## Exécution

1. Assurez-vous d'avoir installé les dépendances nécessaires (`dash`, `dash-bootstrap-components`, `pandas`, etc.).
2. Optionnel : définissez la variable d'environnement `HYPIXEL_API_KEY` avec votre clé API Hypixel.
3. Lancez l'application :

```bash
python main.py
```

Le serveur démarrera sur [http://127.0.0.1:8050/](http://127.0.0.1:8050/) et s'ouvrira automatiquement dans votre navigateur.
